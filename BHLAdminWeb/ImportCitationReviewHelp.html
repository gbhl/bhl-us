Using The Import Segments Review Page

The "Import Segments - Review" page allows the contents of an import file to be reviewed. Individual rows or the entire file can be imported or rejected.

The following actions can be taken on this page:

  • Click on the plus sign at the left of each row to see additional details about that row, including error messages related to the import of that row.
  • Click the page icon next to the Start Page to view the start page of the segment.
  • View and edit the production BHL Author identifiers associated with the segment authors. Authors matched to existing records will include the ID of the production record (i.e. "John Doe (BHL ID 12345)"). Click the "Edit" link next to the author if the matched production record is incorrect.
  • Change the status of individual rows (for more information, see below).
  • Click the "Import File" button to import records with "OK" status into production.
  • Click the "Reject File" button to reject all records with an "OK" status."

The status of each row controls the action that will be taken on the row.

  • Rows with a status of "OK" are ready to be imported. Only rows in this status will be imported.
  • A row with a "Rejected" status will not be imported. This status is set by the person performing the import.
  • A row with an "Error" status failed to import. Check the error messages to determine the reason.
  • A row with an "Warning" status is missing some required data or may have some invalid data, and will not be imported. Can be overridden by setting the status to "OK".
  • A row with a "Duplicate" status duplicates a segment already in BHL, and will not be imported. Can be overridden by setting the status to "OK".
  • A row with an "Invalid" status includes some invalid data, and will not be imported. Check the error messages, correct the data at the source, and re-import.
  • Rows with a status of "Imported" have been imported. This status cannot be changed.

Change the status of any row by clicking on the status of the row. A dropdown of available status values will appear. Pick the desired status from the list.

When the statuses of all rows have been set, take one of the following actions:

  • Import all "OK" rows by clicking the "Import File" button.
  • Reject all "OK" rows by clicking the "Reject File" button.
  • Leave the page without taking any action on the "OK" rows by clicking "Return to Dashboard" or "View Segment Import History" links in the upper left.
