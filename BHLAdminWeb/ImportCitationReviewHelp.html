Using The Import Citations Review Page

The "Import Citations - Review" page allows the contents of an import file to be reviewed. Individual rows or the entire file can be imported or rejected.

The status of each row controls the action that will be taken on the row.

  • Rows with a status of "New" are ready to be imported.
  • A row with a "Rejected" status will not be imported for a business reason. For example, the row may duplicate data already in BHL.
  • A row with an "Error" status will not be imported due to problems with the data.
  • Rows with a status of "Imported" have been imported, and cannot be changed.

Change the status of any row by clicking on the status of the row. A dropdown of available status values will appear. Pick the desired new status from the list.

When the statuses of all rows have been set, take one of the following actions:

  • Import all "New" rows by clicking the "Import File" button.
  • Reject all "New" rows by clicking the "Reject File" button.
  • Leave the page without taking any action on the "New" rows by clicking "Return to Dashboard" or "View Citation Import History" links in the upper left.
