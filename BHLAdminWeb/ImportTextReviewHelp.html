Using The Import Item Text Review Page

The "Import Item Text - Review" page allows the contents of one or more import files to be reviewed. Each file should contain the complete transcription for a single BHL item. Individual files or the entire batch can be imported or rejected.

The following actions can be taken on this page:

  • Click on a file to see a list of the pages in the item.
  • Click on a page description (i.e. "Page 1") to view the page image, the original page text (ocr or transcription), and the replacement page text (transcription).
  • Change the status of individual files (for more information, see below).
  • Click the "Import Batch" button to import all files with "Ready to Import" status into production.
  • Click the "Reject Batch" button to reject the entire batch of files.

The status of each file controls the action that will be taken on the file.

  • Files with a status of "Ready to Import" are ready to be imported. Only files in this status will be imported.
  • A file with a "Rejected" status will not be imported. This status is set by the person performing the import.
  • A file with an "Error" status failed to import. View the error details by clicking the icon next to the file status.
  • A file with a "Must Review" status has already had its text replaced, and the new replacement text must be manually verified before import. Must be overridden by setting the status to "Ready to Import".
  • Files with a status of "Imported" have been imported. This status cannot be changed.

Change the status of any file by clicking on the status of the file. A dropdown of available status values will appear. Pick the desired new status from the list.

When the statuses of all files have been set, take one of the following actions:

  • Import the trsnscriptions in of the all "Ready to Import" files by clicking the "Import Batch" button.
  • Reject all files by clicking the "Reject Batch" button.
  • Leave the page without taking any action on the "Ready to Import" files by clicking "Return to Dashboard" or "View Text Import History" links in the upper left.
