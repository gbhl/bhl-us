About the Library Stats

For each statistic on the administrative dashboard, the "All Records" column shows the total number of records in the BHL database, and the "Active" column shows the number of records that are visible to the public.

Statistic Description
Titles The number of Titles (ex: "The Origin of Species", "Annals of the Missouri Botanical Garden").
Items The number of Items (individual books/volumes).
Pages The number of scanned Pages.
Segments The number of articles/chapters/treatments/ect.
Items w/ Segments The number of Items that contain articles/chapters/treatments/ect.
Names The total number of name strings found.
Unique Names The number of unique name strings.
Verified Names The number of unique and verified name strings.  A verified name is one that has been resolved against a name authority (NameBank, Catalogue of Life, etc).
EOL Names Total number of name strings that are associated with an EOL ID.
Pages with EOL Names (EOL Pages) The number of scanned pages that have at least one name string that is associated with an EOL ID.

The statistics shown on the dashboard are generated in real-time.  That is why an extra click is required to access all of the Name statistics, and why it takes a minute or two to generate each Name statistic.

Note that the statistics shown on the Expanded Library Stats page are NOT generated in real-time.  That page shows statistics that are compiled on a weekly basis, so they may not exactly match the statistics shown on the dashboard.
