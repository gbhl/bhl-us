Item Pagination Statuses

Status Description
NewItem has not been paginated in BHL. It may have been paginated in another tool (i.e. Macaw) before being ingested into BHL.
IncompleteItem has been partially paginated in BHL. The item is not locked, so anyone can continue paginating the item.
In ProgressItem is locked for pagination. Only the user holding the lock can continue paginating the item.
CompleteItem has been paginated in BHL.
