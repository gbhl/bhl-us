GetPageMetadata
Return metadata about a page. You may choose to include the text and a list of names found on the page.
SYNTAX
https://www.biodiversitylibrary.org/api3?op=GetPageMetadata
&pageid=<BHL identifier of a page>
&ocr=<"t" or "true" to include the page text in the response>
&names=<"t" or "true" to include scientific names on the page in the response>
&apikey=<API key value>
EXAMPLE REQUEST
            
https://www.biodiversitylibrary.org/api3?op=GetPageMetadata&pageid=3137380&ocr=t&names=t&apikey=12345678-BBBB-DDDD-FFFF-123456789012
