GetInstitutions
Get a list of institutions which have contributed to BHL in some way. These institutions may fill roles such as contributors, rights holders, and scanning institutions.
SYNTAX
https://www.biodiversitylibrary.org/api3?op=GetInstitutions
&apikey=<API key value>
EXAMPLE REQUEST
            
https://www.biodiversitylibrary.org/api3?op=GetInstitutions&apikey=12345678-BBBB-DDDD-FFFF-123456789012
