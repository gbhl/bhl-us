NameSearch
Search for a particular name. Names both with and without NameBank identifiers are returned.
SYNTAX
https://www.biodiversitylibrary.org/api3?op=NameSearch
&name=<a partial or complete name string>
&apikey=<API key value>
EXAMPLE REQUEST
            
https://www.biodiversitylibrary.org/api3?op=NameSearch&name=poa+annua&apikey=12345678-BBBB-DDDD-FFFF-123456789012
