SubjectSearch
Return a list of subjects that match (fully or partially) the specified search string.
SYNTAX
https://www.biodiversitylibrary.org/api3?op=SubjectSearch
&subject=<the full or partial subject for which to search>
&format=<"xml" for an XML response or "json" for JSON (OPTIONAL; "xml" is the default)>
&apikey=<API key value>
EXAMPLE REQUEST (XML)
            
https://www.biodiversitylibrary.org/api3?op=SubjectSearch&subject=diptera&format=xml&apikey=12345678-BBBB-DDDD-FFFF-123456789012
