AuthorSearch
Return a list of authors that match (fully or partially) the specified search string. The names searched are those contained in MARC 100a, 110a, 111a, 700a, 710a, and 711a library records.
SYNTAX
https://www.biodiversitylibrary.org/api3?op=AuthorSearch
&authorname=<full or partial name of the author for which to search>
&format=<"xml" for an XML response or "json" for JSON (OPTIONAL; "xml" is the default)>
&apikey=<API key value>
EXAMPLE REQUEST (XML)
            
https://www.biodiversitylibrary.org/api3?op=AuthorSearch&authorname=darwin&format=xml&apikey=12345678-BBBB-DDDD-FFFF-123456789012
