GetLanguages
Get a list of languages in which materials in BHL have been written.
SYNTAX
https://www.biodiversitylibrary.org/api3?op=GetLanguages
&format=<"xml" for an XML response or "json" for JSON (OPTIONAL; "xml" is the default)>
&apikey=<API key value>
EXAMPLE REQUEST (XML)
            
https://www.biodiversitylibrary.org/api3?op=GetLanguages&format=xml&apikey=12345678-BBBB-DDDD-FFFF-123456789012
